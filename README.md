SYSTEMSTORES – Online Shopping Platform
Overview
SYSTEMSTORES is a modern, responsive online shopping interface designed to showcase featured products, provide a smooth browsing experience, and support multilingual content.
The project uses HTML5, CSS, Bootstrap 5, and a custom translation system powered by data-translate attributes.
This README describes the structure, features, and setup of the project.

Features
🛒 E‑Commerce Layout
- Homepage with featured products
- Product comparison table
- Navigation links to:
- Products
- Cart
- Contact
- About Us
- Order History
- Account
- Login
🌍 Multilingual Support
The page uses a custom translation system based on:
data-translate="KEY"


A language switcher allows users to switch to French.
📱 Responsive Design
- Built with Bootstrap 5.3.3
- Mobile‑friendly navigation bar
- Responsive product grid
🛍 Product Display
Each product includes:
- Image
- Title
- Price
- Description
📊 Product Comparison Table
A clean table comparing:
- Product name
- Price
- Comments
📞 Contact Options
Footer includes:
- Phone link
- Email link
- Privacy Policy
- Language switcher
